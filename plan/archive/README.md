# `plan/archive/`

Historical record. These were the daily `checkpoint-YYYY-MM-DD-eod.md`
files from the pre-windup era — exhaustive day-end snapshots written
when "what state is the repo in" was carried session-to-session via
those docs.

The pattern was superseded on 2026-05-24 when the `/windup` skill
made `plan/YYYY-MM-DD-summary.md` the canonical day-end doc (tight,
bulleted, signal-not-prose). See CLAUDE.md §"Documentation cadence"
and the `W1.9` entry in pending.md.

Existing checkpoints moved here on 2026-05-29; nothing new is written
to this folder. If a reference in another doc points at a path like
`plan/checkpoint-*-eod.md`, the file lives here.
